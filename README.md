JasmineTest
==========

Dependency
-------------------------
- nodejs
- npm

Installation
-------------------------
$ npm install jasmine-node --save

NOTE : --save : to add the installed package to the package.json file

Run Test
-------------------------
./node_modules/.bin/jasmine-node spec
